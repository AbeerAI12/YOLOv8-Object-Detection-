# YOLOv8-Object-Detection

## Project Overview
* **Goal:** Implement real-time object detection using the YOLOv8 architecture.
* **Techniques:** Computer Vision, deep learning-based object localization, and inference.
* **Tools:** Python, Ultralytics YOLOv8, OpenCV.

## Key Features
* **Real-time Detection:** Utilizing a high-performance model for fast and accurate object identification.
* **Model Inference:** Processing images/videos to detect and annotate objects with bounding boxes.
* **Computer Vision Pipeline:** Integrating model outputs with image processing workflows.

## How to Run
1. Clone this repository.
2. Install the required dependencies: `pip install ultralytics opencv-python`.
3. Load the YOLOv8 model and run inference on your test images or video stream.
